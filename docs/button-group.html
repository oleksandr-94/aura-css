Group

Joins adjacent controls — buttons and form fields — into one segmented unit with square inner corners and a single shared border. Wrap any mix of .btn, .input, .select.

CategoryClassEffect
Componentgroup (alias btn-group)Joins its direct children
Children.btn · .input · .selectFields grow, buttons stay natural
Modifiergroup--blockFull width, equal items
Radiosegmented · segmented__optionSingle-select via radios (no JS)

Button group

<div class="group">
  <button class="btn btn--ghost">Day</button>
  <button class="btn btn--ghost">Week</button>
  <button class="btn btn--ghost">Month</button>
</div>

Radio segmented single-select · no JS

When exactly one option is active, use .segmented — native radios, so it's keyboard-accessible (arrow keys) and submits with a form. The checked pill is skin-aware.

<div class="segmented" role="radiogroup" aria-label="View">
  <label class="segmented__option">
    <input type="radio" name="view" checked><span>Day</span>
  </label>
  <label class="segmented__option">
    <input type="radio" name="view"><span>Week</span>
  </label>
  <label class="segmented__option">
    <input type="radio" name="view"><span>Month</span>
  </label>
</div>

Input group

<div class="group">
  <input class="input" type="text" placeholder="Search projects…">
  <button class="btn">Search</button>
</div>
