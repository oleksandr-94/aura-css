Button

Buttons trigger an action. The base .btn is the primary button; add modifiers for colour, style, and size. Their surface (glow / flat / extruded) follows the active skin — try the Skin switch on the left.

Gradient

A two-colour blend. On its own, .btn--gradient is the brand --primary → --secondary. Combine it with any colour modifier and it blends --<role> → --<role>-2 (defined in the theme; falls back to a mono gradient of the single colour). Set --on-gradient in a theme to force the text colour on gradients only.

<button class="btn btn--gradient">Primary</button>
<button class="btn btn--gradient btn--success">Success</button>
<button class="btn btn--gradient btn--error">Error</button>
