Usage with React / Vue

Aura is CSS-first: components are class names plus a data-state attribute. In a framework you own the state and bind data-state to it — the JS layer is only for toasts and a few pure helpers.

ℹ
Rule of thumb
Don't use mount / open / close / toggle in a framework — they write data-state straight to the DOM and desync from your component state. Bind data-state yourself; import only toast, trapFocus, lockScroll.

Install

import 'aura-css/css';                       // glass skin (or aura-css/flat, aura-css/neu)
import { toast, trapFocus, lockScroll, unlockScroll } from 'aura-css';

React — Modal

import { useEffect, useRef } from 'react';
import { trapFocus, lockScroll, unlockScroll } from 'aura-css';

function Modal({ open, onClose, children }) {
  const ref = useRef(null);

  useEffect(() => {
    if (!open) return;
    lockScroll();
    const untrap = trapFocus(ref.current);
    const onKey = (e) => e.key === 'Escape' && onClose();
    document.addEventListener('keydown', onKey);
    return () => { untrap(); unlockScroll(); document.removeEventListener('keydown', onKey); };
  }, [open, onClose]);

  return (
    <div className="modal-overlay" data-state={open ? 'open' : 'closed'} data-overlay
         onClick={(e) => e.target === e.currentTarget && onClose()}>
      <div className="modal" role="dialog" aria-modal="true" ref={ref}>{children}</div>
    </div>
  );
}

React — Dropdown

function Dropdown() {
  const [open, setOpen] = useState(false);
  const ref = useRef(null);

  useEffect(() => {
    if (!open) return;
    const onDoc = (e) => !ref.current.contains(e.target) && setOpen(false);
    document.addEventListener('click', onDoc);
    return () => document.removeEventListener('click', onDoc);
  }, [open]);

  return (
    <div className="dropdown" data-state={open ? 'open' : 'closed'} ref={ref}>
      <button className="btn" onClick={() => setOpen(v => !v)}>Menu <span className="dropdown__caret" /></button>
      <div className="dropdown__menu">
        <a className="dropdown__item" href="#">Profile</a>
        <a className="dropdown__item" href="#">Sign out</a>
      </div>
    </div>
  );
}

Toasts — the exception

Toasts are imperative by nature, so toast() is exactly what you want — call it from any handler.

import { toast } from 'aura-css';

<button className="btn" onClick={() => toast('Saved!', { type: 'success' })}>
  Save
</button>

Vue

<template>
  <div class="modal-overlay" :data-state="open ? 'open' : 'closed'" data-overlay
       @click.self="open = false">
    <div class="modal" role="dialog" aria-modal="true" ref="dialog">
      <slot />
    </div>
  </div>
</template>

<script setup>
import { watch, ref } from 'vue';
import { trapFocus, lockScroll, unlockScroll } from 'aura-css';
// bind :data-state to `open`; call trapFocus/lockScroll in a watcher.
</script>

What to import

From auraIn React / Vue
toast()✅ call from any handler
trapFocus · lockScroll · unlockScroll✅ in effects / watchers
configure()✅ if you set a prefix
mount · open · close · toggle❌ vanilla only — they write to the DOM past your state
