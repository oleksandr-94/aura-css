Toast

A transient notification. Show it imperatively with the JS helper toast(message, { type, timeout }), or render the markup yourself and flip data-state.

CategoryClass / APIEffect
Containertoast-container --bottomFixed stack (auto-created)
ComponenttoastOne notification
Colortoast--success --warning --error --infoAccent
APItoast(msg, opts)Create + auto-dismiss

Trigger a toast

import { toast } from 'aura-css/interactions';   // or window.Aura.toast

toast('Changes saved.', { type: 'success' });
toast('Something went wrong.', { type: 'error', timeout: 6000 });

Static markup

Changes saved successfully.
<div class="toast toast--success" data-state="open">
  <div class="toast__body">Changes saved successfully.</div>
  <button class="toast__close">×</button>
</div>
