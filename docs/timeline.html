Timeline

A vertical feed of events. A single rail runs down the left with an accent dot on every item; the --now item glows and --muted items dim. Item bodies sit on --surface-2 and nudge on hover.

CategoryClassEffect
ComponenttimelineVertical event list with a rail
Layouttimeline--rowHorizontal items: time | body | count
Parttimeline__itemOne event (accent dot on the rail)
Statetimeline__item--nowCurrent event (brand-coloured glowing dot; override via inline --_c)
Statetimeline__item--mutedDe-emphasised event
Parttimeline__timeTimestamp line
Parttimeline__title timeline__subTitle + subtitle
Parttimeline__countTrailing count pill
