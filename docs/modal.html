Modal

A dialog driven by data-state. Mark the overlay data-overlay and the optional JS adds scroll-lock, focus-trap, Esc and backdrop-close. In frameworks, bind data-state to your own state instead.

Componentmodal-overlay + modalBackdrop + dialog

<script type="module" src="interactions.auto.js"></script>

In a framework (React)

// No library JS needed — bind data-state to your own state.
<div className="modal-overlay" data-state={open ? 'open' : 'closed'} data-overlay>
  <div className="modal" role="dialog" aria-modal="true">
    <h3 className="modal__title">Delete workspace?</h3>
    <div className="modal__actions">
      <button className="btn btn--ghost" onClick={() => setOpen(false)}>Cancel</button>
    </div>
  </div>
</div>
// optional a11y helpers: import { trapFocus, lockScroll } from 'aura-css/interactions'
