Stat

A metric card: icon, big value, and label. The surface is rendered through the active skin (glass / flat / neu), exactly like .card. Colour variants add a top accent bar and a faint tint pulled from the same colour.

Horizontal (stat--row)

Add stat--row and wrap the value and label in stat__body so the text sits as one block to the right of the icon. Colour variants (bar + tint) work the same. Use stat__icon--plain for a transparent slot and --_isize to grow the chip.
